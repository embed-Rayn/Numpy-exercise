## Numpy
Numpy는 아래와 같은 특징을 갖습니다.

- 수치해석용 Python 라이브러리
- CPython에서만 사용 가능
- BLAS/LAPACK 기반
- ndarray 다차원 행렬 자료 구조 제공
- 내부 반복문 사용으로 빠른 행렬 연산 가능
- 행렬 인덱싱(array indexing) 기능
